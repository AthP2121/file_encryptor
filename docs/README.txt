Thank you for installing File Encryptor!

OVERVIEW:
---------
File Encryptor is a simple, secure application for encrypting and decrypting
files and folders using AES-256 encryption. It supports both password-based
encryption and key file encryption.

QUICK START:
------------
1. Launch File Encryptor from the Start Menu or Desktop shortcut
2. Click "Add Files" or "Add Folder" to select what to encrypt
3. Choose "Password" or "Key File" mode
4. Enter a strong password (or generate/load a key file)
5. Click "Encrypt Selected"
6. Your files will be encrypted with a .locked extension

TO DECRYPT:
-----------
1. Select .locked files in the application
2. Enter the same password or load the same key file
3. Click "Decrypt Selected"
4. Your original files will be restored

SECURITY NOTES:
---------------
✓ Uses AES-256 encryption (industry standard)
✓ PBKDF2 key derivation with 480,000 iterations
✓ Each file has a unique salt

⚠ IMPORTANT:
- Remember your passwords! There is NO password recovery
- Keep key files safe! Losing them means losing your data
- Make backups before encrypting important files
- Test decryption before deleting originals

FEATURES:
---------
• Encrypt individual files or entire folders
• Batch encrypt/decrypt multiple files
• Search for files to encrypt
• Password or key file authentication
• Progress tracking for batch operations
• Automatic folder compression

SYSTEM REQUIREMENTS:

• ~50 MB disk space
• No additional software required
